"Don't buy things with money, buy them with time."

Time is Money is a web-based financial decision tool that shifts your perspective from currency to time. By calculating how many hours, days, or months of your life are exchanged for a purchase, it helps users make more conscious and impactful financial decisions.

## âœ¨ Features

* Real-time Calculation: Instant results while typing, no page reload needed.
* Life Cost Perspective: Converts monetary prices into Months, Weeks, Days, and Hours of work.
* Global Currency Support: Supports major global currencies (USD, EUR, BRL, JPY, GBP, etc.).
* Flexible Income Inputs: Calculates based on Monthly, Annual, or Hourly income.
* Internationalization (i18n): Interface fully translated into English, Spanish, and Portuguese.
* Privacy First: Everything runs in the browser. No personal data is stored or transmitted.
* Modern Design: Clean, responsive "Glassmorphism" interface (White Mode) built with Tailwind CSS.
* Data Visualization: Interactive chart (Chart.js) to visualize the temporal distribution of the cost.
